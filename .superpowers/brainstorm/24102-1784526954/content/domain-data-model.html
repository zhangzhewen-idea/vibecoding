原始文件

/Users/zhangzhewen/data/ragdemo/uploads/{knowledgeBaseId}/{uuid}.{ext}。数据库只保存路径和元数据，不保存 BLOB；文件名使用 UUID，原始名称单独记录，避免重名和路径穿越。

一致性规则：向量化成功后文档才进入 READY；失败记录原因并允许管理员重试。删除文档时先删除 Redis 切片，再删除物理文件，最后将 MySQL 文档记录逻辑删除；任何一步失败都保留可重试状态。
