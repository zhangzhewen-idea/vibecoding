第三节：前端信息架构与页面职责

智能紫渐变作为品牌风格，正文统一黑色；管理员拥有全部用户能力，普通用户只进入问答工作台。

PUBLIC

/login

项目名称、紫蓝视觉背景
账号密码登录
演示账号提示

USER WORKSPACE

/home

欢迎信息、可用知识库、最近会话、快捷提问

/knowledge-bases

只展示已启用知识库；卡片查看简介并创建会话

/chat/:conversationId

会话侧栏、流式答案、停止生成、引用角标与来源抽屉

/conversations

本人历史会话查询、重命名和删除

/profile

个人资料；暂不提供密码修改，保持 MVP 范围

ADMIN CONSOLE

/admin/dashboard

4 个指标卡、7 日趋势、文件类型环图、热门知识库 Top 5

/admin/knowledge-bases

新建、编辑、启停、封面、进入文档管理

/admin/documents

上传、状态进度、失败原因、重试向量化、删除

/admin/users

新增用户、分配角色、启停、重置为 123456

用户问答入口

管理员同时可访问全部 USER WORKSPACE 页面

状态反馈

骨架屏、空状态、上传进度、处理状态标签、失败原因与重试按钮。

响应式范围

优先桌面端 1280px+；兼容平板，不专门设计移动端管理后台。

图表实现

使用 Apache ECharts；只展示真实 SQL 聚合数据，不生成随机统计数。

✓

批准这一节

确认路由、角色页面、交互状态、桌面端优先和 ECharts 统计范围。
